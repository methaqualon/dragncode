Drag'n'code


Puppeteer code generator via drag'n'dropping blocks from the left to the right



Fully client-side


Don't require any server-side logic, just only one HTML file with scripts and styles included via CDN.
Keep in mind that it don't have any initialization code, only `payload`.

Very simple


Just move any blocks from the left panel to the right panel, and code will be generated.
Change any options in clonned blocks.
Take generated code.

Extendable


Download repository to your PC and edit this file if you want to add/edit any commands.
Feel free to communicate with us - drag'n'code's github
